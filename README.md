# Student Management System (Java Swing)

## Description
This is a simple Student Management System built using Java Swing.
It allows users to add, view, and delete student records through a graphical user interface (GUI).

## Features
- Add student (ID & Name)
- View all students
- Delete student by ID
- Simple and user-friendly GUI

## Technologies Used
- Java
- Swing (GUI)
- AWT
- Event Handling
- ArrayList

## How to Run
1. Open the project in Eclipse IDE
2. Run the `Student` class
3. Enter Student ID and Name
4. Use buttons:
   - Add → to add student
   - View → to display all students
   - Delete → to remove student

## Output
A GUI window will open where you can manage student records.
